Slimbox
=======

See [Slimbox's homepage](https://rok.strnisa.com/slimbox/) for more info.

License
-------
See [LICENSE](LICENSE).

Dependencies
------------
* Install Java 11.
* Install Maven.

Build
-----
* Run `mvn clean package`.

Run
---
* Run `java -jar target/slimbox-beta-1.6.0-jar-with-dependencies.jar`.
